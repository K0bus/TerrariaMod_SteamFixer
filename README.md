# Steam Achievement Fixer

**Steam Achievement Fixer** is a tModLoader mod for Terraria that helps synchronize achievements across tModLoader and Terraria Steam achievements

---

## Features

- Reads **Steam achievements** via SteamID64 and the Steam XML API (public profile only).
- Handles **Terraria vanilla achievements** within the mod.
- Provides the `/pushallachievements` command to push all achievements to steam.
- Provides the `/pullsteamachievements` command to fetch all steam achievements.
